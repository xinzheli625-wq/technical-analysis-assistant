# Trend Analysis Reference

## Wyckoff Stage Analysis

Stocks cycle through 4 stages. Stage 2 is the only stage for new long positions.

| Stage | Name | Characteristics | Action |
|-------|------|----------------|--------|
| 1 | Basing | Price near 200MA, MA flat, volume declining | Watch, wait |
| 2 | Advancing | Higher highs/lows, bullish MA alignment, volume on up days | Buy zone |
| 3 | Topping | Wide swings, false breakouts, heavy volume without progress | Reduce |
| 4 | Declining | Below all MAs, bearish alignment | Avoid |

### Base Counting (within Stage 2)

| Base # | Position Quality | Size Guidance |
|--------|-----------------|---------------|
| 1-2 | Best, most upside | Full position |
| 3-4 | Still valid | Reduce 25% |
| 5-6 | Late stage | Reduce 50% |
| 7+ | Likely topping | Avoid |

## Moving Average Alignment

**Bullish Staircase (mandatory for buy):**
```
Price > 50MA > 150MA > 200MA (and 200MA rising)
```

**Bearish Staircase:**
```
Price < 50MA < 150MA < 200MA (and 200MA falling)
```

**Partial alignment signals:**
- Price > 50MA but < 200MA: Short-term bullish, intermediate caution
- 50MA crossing above 150MA: "Golden cross" intermediate signal

## Trend Line Rules

- A trend line needs at least 3 touches to be valid
- The more touches, the more significant the level
- A break of a 3+ touch trend line is a significant signal
- Volume should confirm the break (higher on break direction)
