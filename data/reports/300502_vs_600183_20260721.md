## 一、结论先行

**1. 行业还有机会吗？——有，但属于"等待确认型"机会，不是"当下抄底型"机会。**

基本面逻辑（AI 资本开支 → 光模块/PCB 需求）未被这波回撤破坏：覆铜板处在涨价周期且中报大面积预增（生益预告 +117%~131%），光模块 1.6T 放量在即、中报预告未来数日披露。技术面上，两家公司**都进入了"空头衰竭观察期"，但都还没有右侧确认信号**——动作是盯触发条件，不是接飞刀。

**2. 光模块 vs PCB，现在谁更被技术面青睐？——短期青睐光模块（结构更成熟），中期 PCB 弹性更大（但需先确认底部）。**

| | 新易盛（光模块） | 生益科技（PCB/覆铜板） |
|---|---|---|
| 流水线方向判断 | **NEUTRAL**（置信度 40%） | **BEARISH**（置信度 75%，但定性为"衰竭期赶底"） |
| 当前阶段 | **区间震荡**（下沿 450-490 / 上沿 560-580） | **下跌趋势末期，恐慌性抛售赶底** |
| 真实回撤（前复权） | 618.87 → 510.0（**-17.6%**，7-01 高点） | 191.88 → 123.34（**-35.7%**，6-22 高点） |
| 20 日涨跌 | -12.1% | -33.1% |
| 多空概率评估 | 多空冲突无主导（触发 Skill 多 10/空 5） | 看涨反转 45% / 延续寻底 45% / 不确定 10% |
| 7-20 单日 | **+5.62%**（区间中枢放量阳线） | **-6.77%**（恐慌抛售，命悬 119 前低） |

- **新易盛**：跌得早、跌得浅（除权修正后真实回撤只有 -17.6%，不是此前误判的 -38%），已经在 490-541 之间横盘了近一个月，7-20 的 +5.62% 是回调以来首次站上区间中枢的放量阳线。**结构最清晰**：上有 518.2/541.37 触发位、下有 490.12/481.5 失效位，适合等信号执行。
- **生益科技**：跌得晚、跌得急（一个月 -35.7%），但 LLM 四阶段裁决已从"趋势延续远未达成"修正为"**抛售高潮、空头色厉内荏**"——极端超卖（KDJ-J 9.3、CCI -174.5、跌破布林下轨）+ ADX 高位回落 + MACD 柱收缩。这是弹性最大的阶段，也是确定性最低的阶段：一旦 119 附近缩量止跌出反转 K 线，反弹会很暴力；跌破 119.06 则打开到 110 的空间。

**一句话：稳健视角选光模块（新易盛结构成熟、触发位明确）；左侧博弈视角 PCB（生益）弹性更大但必须先等底部确认信号。两者的"青睐"都是观察名单级别，还不是买入级别。**

## 二、行业背景速览（2026 年 7 月）

### 光模块（新易盛所在）
- 光模块三巨头（中际旭创、新易盛、天孚通信）自 2026 年 6 月高点回调 13%-16%，回撤主因：一季报环比 -13.25% 引发的预期修正 + 高位获利盘兑现 + 估值消化，而非 AI 需求逻辑逆转。
- 基本面支撑仍在：2026Q1 营收同比 +106%，1.6T 光模块 2025 下半年量产、2026 加速出货，泰国工厂二期扩产，预付款环比暴增锁定上游物料。
- 分歧点：远期（2026-2027）供给过剩担忧、800G/1.6T 年降压力、EML 光芯片与 PCB 物料紧缺。
- **关键催化剂：7 月中下旬的半年报业绩预告**（尚未披露）——超预期则修复，不及预期则测试区间下沿。
- 公司行动：2026-06-11 实施 2025 年度分红（10转4派10元），2025 年营收 +187%、净利 +236%。

### PCB / 覆铜板（生益科技所在）
- 生益科技 7-13 晚发布业绩预告：2026H1 归母净利 30.99-32.98 亿元，**同比 +117%~131%**；7-13 跌停、7-14 涨停、随后五个交易日又从 148 跌到 123——**利好落地变成资金撤退的出货窗口**，博弈极其剧烈。
- 行业层面：A股 32 家 PCB 产业链公司 28 家预增、近六成净利翻倍；覆铜板二季度四轮提价，FR4 价格翻倍，中高端 CCL 交期从 2 周拉长到 6 周；上游覆铜板企业平均净利增速 187%。
- 核心逻辑：深度绑定英伟达供应链（M9 级覆铜板用于 Rubin 架构 78 层正交背板），AI 服务器材料等级从 M4 跃升至 M6-M9。
- 风险：利好兑现后资金持续撤退（7-20 成交额放大下跌）、涨价持续性、原材料成本。

## 三、新易盛（300502）技术面完整诊断（前复权）

**流水线执行**：触发 Skill 40 条 / 准触发 121 条 ｜ 市场环境：**volatile（震荡 + 波动扩张）**，置信度 0.65 ｜ 记录ID：`875ee7ef`

### 四阶段结论（Phase 4）
- **方向：NEUTRAL**，置信度 **40%**
- 趋势性质：**ranging（区间震荡）**——原上升趋势已衰竭（ADX=14.8 无趋势状态），大幅回调后进入宽幅震荡，**区间下沿 450-490，区间上沿 560-580**。
- 核心证据：价格在 SMA20（541.37）与 SMA50（513.68）之间震荡；MACD 死叉但柱收缩，空头动能减弱；多空信号冲突无主导（Phase 3）。

### 关键价位
| 类型 | 价位 | 依据 |
|---|---|---|
| 看多触发 | **518.2** | 突破最近阻力则短多成立 |
| 趋势转多确认 | **541.37 / 535.91** | SMA20 / 一目基准线，放量站上才谈反转 |
| 区间中枢 | 513.68 | SMA50，7-20 收盘 510 贴身 |
| 看空触发 | **490.12** | 跌破最近支撑 |
| 判断失效 | **481.5** | 7-20 低点，跌破则空头加强 |

### 观察点
1. ADX 能否回升至 20 以上（趋势重启信号）
2. MACD 能否金叉（短期多空转换）
3. 能否站上 SMA20（541.37）
4. OBV 能否止跌回升（资金回流确认；当前价格-OBV 潜在背离是最大隐忧）
5. **基本面：中报业绩预告（未来数日）**

### 判断失效条件
- 跌破 481.5 → 空头趋势加强
- 放量站上 535.91 → 看多信号增强
- ADX 跌破 10 → 极弱趋势，完全观望

### 风险提示
- 历史波动率 102.7%，止损易被频繁触发；ADX 低位时任何突破都可能是假突破。

## 四、生益科技（600183）技术面完整诊断

**流水线执行**：触发 Skill 45 条 / 准触发 93 条 ｜ 市场环境：**trending_down（下跌趋势，早期）**，置信度 0.65 ｜ 记录ID：`fd60b8e2`

### 四阶段结论（Phase 4）
- **方向：BEARISH**，置信度 **75%**
- 趋势性质：**fading（趋势末期加速赶底）**——主要下跌趋势进入衰竭期，当前是典型的恐慌性抛售；趋势"生命"正快速消耗，反弹随时可能发生且剧烈。
- 多空概率（Phase 3）：**看涨反转 45% / 延续寻底 45% / 不确定 10%**——"抛售高潮"阶段，牛熊转折的关键观察期。
- 空头证据链：价格在所有均线之下 + 多框架一致看跌 + 放量下跌 OBV 流出；但 ADX 下降、MACD 收缩、极端超卖揭示空头"色厉内荏"。

### 关键价位
| 类型 | 价位 | 依据 |
|---|---|---|
| 多空分水岭 | **119.06** | 前低支撑，跌破则看 110 整数关口；守住并出反转信号则底部成立 |
| 短期改善触发 | **127.72** | 站稳 S1 枢轴点则极度看空转为震荡 |
| 空头止损参考 | 131 | 收复 7-20 阴线一半 |
| 下跌目标 | 110 | 心理关口（LLM 特别注明：此处不做空头目标，风险回报比已差） |

### 观察点（抛售高潮是否结束的确认清单）
1. 次日开盘：高开、或大幅低开后快速拉回 + 缩量
2. K 线：长下影锤子线 / 十字星 / 看涨吞没
3. 120 附近能否企稳 + MACD 柱翻红 / RSI 上穿 50（底背离确认）
4. **基本面：中报正式披露与覆铜板第八轮提价能否落地**

### 判断失效条件
- 缩量小实体在 120-125 盘整数日 + 放量阳线突破 → 转 NEUTRAL/BULLISH
- 跌穿 119.06 后巨量快速拉回（弹簧效应/空头陷阱）→ BEARISH 失效
